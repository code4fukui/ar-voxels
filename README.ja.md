# AR Voxels

AR 環境で 3D ボクセルを作成・操作できる Webアプリケーションです。

## デモ
https://code4fukui.github.io/ar-voxels/

## 機能
- マウスやジョイスティックでカメラを操作できます
- Shiftキーを押しながらクリックするとボクセルを削除できます
- 数字キーを押してブラシの色を変更できます

## 必要環境
WebXRに対応したブラウザが必要です

## 使い方
1. リポジトリをクローンする
2. `index.html`をブラウザで開く
