# AR Voxels

AR Voxels is a web-based augmented reality application that allows users to create and manipulate 3D voxel structures in a virtual environment.

## Demo
You can view the live demo at https://code4fukui.github.io/ar-voxels/

## Features
- Create and place 3D voxel cubes in a virtual space
- Change the color of the voxel brush
- Interact with the voxel structure using mouse controls
- WebXR support for AR/VR experiences

## Requirements
This project relies on the following libraries:
- Three.js
- egxr.js
- xrutil

## Usage
1. Clone the repository:
```
git clone https://github.com/code4fukui/ar-voxels.git
```
2. Open the `index.html` file in a web browser.

## Data / API
This project does not use any external data or APIs.
